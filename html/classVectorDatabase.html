Detailed Description

Класс, реализующий простую векторную базу данных с возможностью сохранения на диск.

Поддерживает добавление эмбеддингов, поиск по схожести (косинусное расстояние), управление метаданными и загрузку/сохранение состояния в файл.

Находит K наиболее похожих записей по отношению к заданному запросу (эмбеддингу).

Поиск осуществляется на основе косинусного сходства. Результат сортируется по убыванию сходства.

Parameters
queryЭмбеддинг запроса.
kКоличество возвращаемых записей (по умолчанию 5).
similarity_thresholdПорог сходства: возвращаются только записи с сходством ≥ порога (по умолчанию 0.0).
Returns
Вектор пар: (ID записи, сходство). Сходство ∈ [–1, 1].
